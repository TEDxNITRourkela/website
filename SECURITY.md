# Security Policy

## Supported Versions

- Version 1.0.x is yet to be released.
- Version 0.x.x are initial development versions.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

- To report a vulnerability, create an issue with the label "vulnerability".
